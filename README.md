## Algoscale Assignment to make fullstack SPA.

front end :
**import**
- react
- react-router-dom
- react-facebook-login
- react-google-login
- react-chartjs-2
- react-notifications
- axios

back end :
**import**
- express
- cors
- mongoose
- nodemon

To run backend run :
- ``` mongod ``` in one terminal.
- go to directory ``` backend ``` and run ``` node index.js ```
- set up your database.

To run front end run:
- ``` npm start ```
